# 第五章 数据可视化
- 使用plot()函数绘制基础图形（散点图、线图），掌握设置标题（main）、坐标轴标签（xlab/ylab）、点形状（pch）、线条类型（lty）、颜色（col）及坐标轴范围（xlim/ylim）的方法
